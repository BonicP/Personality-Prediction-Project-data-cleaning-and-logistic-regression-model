# Behaviour Analysis – Data Cleaning and Logistic Regression

Este projeto tem como objetivo demonstrar um fluxo completo de trabalho em ciência de dados, desde a análise exploratória e limpeza de dados até ao treino e avaliação de um modelo de machine learning.

## Objetivo
Realizar a preparação e análise de um conjunto de dados comportamentais, aplicando técnicas de limpeza de dados e visualização, e treinar um modelo de regressão logística para fins de classificação.

## Tecnologias Utilizadas
- Python  
- Pandas e NumPy  
- Matplotlib e Seaborn  
- scikit-learn

## Principais Etapas
- Análise exploratória dos dados (EDA)  
- Identificação e tratamento de valores em falta e inconsistências  
- Preparação dos dados para modelação  
- Treino e avaliação de um modelo de regressão logística  
- Visualização de resultados

## Notas
Este projeto foi desenvolvido no contexto de aprendizagem em ciência de dados.  
O modelo foi treinado num conjunto de dados limitado, pelo que os resultados devem ser interpretados como demonstrativos do processo e não como um modelo final de produção.
